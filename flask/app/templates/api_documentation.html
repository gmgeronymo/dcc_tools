Introdução

API DCC Tools

DCC Generator

A API recebe como entrada um arquivo em formato JSON.

Exemplo de arquivo JSON:
{ "chave1": "valor", "chave2": "valor", "chave3": [ { "subchave1": "valor", "subchave2": "valor" } ], "chave4": "valor" }

Descrição das Chaves

Chave Descrição Tipo Exemplo Observações

num_processo String com o número do processo Obrigatória "num_processo": "0052600.003988/2023-06"
tipo_item String com a identificação do item Obrigatória "tipo_item": "Padrão de Transferência AC-DC"
fabricante String com o fabricante do item Obrigatória "fabricante": "Fluke"
modelo String com o modelo do item Obrigatória "modelo": "792A"

desc_tecnico_executor Descrição do cargo do técnico executor Obrigatória "desc_tecnico_executor": "Técnico Executor"

Software

Array com informações do software utilizado para gerar o DCC

Tipo: Opcional

Exemplo:
"software": [ { "name": "SYS-LAMPE", "version": "v2.3-69-3526f2d" } ]
Descrição do array "software":
Chave Descrição Tipo Exemplo
name Nome do software Obrigatória "name": "SYS-LAMPE"
version Versão do software Obrigatória "version": "v2.3-69-3526f2d"

RefTypeDefinitions

Array com definições de refTypes adicionais

Tipo: Opcional

Documentação: https://digilab.ptb.de/dkd/refType/vocalo/index.php?tema=2&/basic

Exemplo:
"refTypeDefinitions": [ { "name": "Namespace for mathematical refTypes", "description": "Namespace for mathematical refTypes used across various domains, such as mass, force, etc., including minimum, maximum, and similar values.", "namespace": "math", "link": "https://digilab.ptb.de/dkd/refType/vocalo/index.php?tema=292" } ]
Descrição do array:
Chave Descrição Tipo Exemplo
name Nome do namespace Obrigatória "name": "Namespace for mathematical refTypes"
description Descrição do namespace Obrigatória "description": "Namespace for mathematical refTypes used across various domains..."
namespace String identificadora do namespace Obrigatória "namespace": "math"
link Link para a definição do namespace Obrigatória "link": "https://digilab.ptb.de/dkd/refType/vocalo/index.php?tema=292"

Informações Pertinentes

Array com informações pertinentes à calibração

Tipo: Obrigatória (o conteúdo é opcional)

Exemplo:
"informacoes_pertinentes": [ { "name": "Temperatura", "value": "23.2", "unc": "1.0", "k": "2", "unit": "\degree eeeelsius", "refType": "basic_temperature" }, { "name": "Umidade relativa", "value": "49", "unc": "10", "unit": "\percent", "refType": "basic_humidityRelative" }, { "name": "Exemplo textual", "text": "Exemplo de informação pertinente textual." } ]
Descrição do array:
Chave Descrição Tipo Exemplo
name Nome da informação pertinente Obrigatória "name": "Temperatura"
unit Unidade da informação (formato D-SI) Obrigatória se for numérica "unit": "\degreecelsius"
value Valor numérico Obrigatória se unit existir "value": "23.2"
unc Valor da incerteza Obrigatória se unit existir "unc": "1.0"
k Fator de abrangência Opcional (padrão: 2) "k": "2"
text Informação textual Obrigatório se unit não existir "text": "Exemplo textual"
refType RefType da informação Opcional (recomendado) "refType": "basic_temperature"

Rastreabilidade

declaracao_rastreabilidade

Campo textual com a declaração da rastreabilidade metrológica

Tipo: Obrigatória

"declaracao_rastreabilidade": "Os resultados da calibração são rastreados ao Sistema Internacional de Unidades (SI), por intermédio dos padrões metrológicos nacionais. As medições realizadas estão referenciadas aos padrões relacionados."
tabela_rastreabilidade

Array com os padrões utilizados

Tipo: Opcional

Exemplo:
"tabela_rastreabilidade": [ { "name": "Conversor Térmico PMJTC", "origem": "PTB", "certificado": "PTB 27812/2020", "cod_id": "PR 394" }, { "name": "Conversor Térmico PMJTC", "origem": "PTB", "certificado": "PTB 27811/2020", "cod_id": "PR 365" } ]
Descrição do array:
Chave Descrição Tipo Exemplo
name Nome do padrão Obrigatória "name": "Conversor Térmico PMJTC"
origem Origem do padrão Obrigatória "origem": "PTB"
certificado Certificado do padrão Obrigatória "certificado": "PTB 27812/2020"
cod_id Código de identificação Obrigatória "cod_id": "PR 394"

Incerteza

declaracao_incerteza

Campo textual com a declaração da incerteza de medição

Tipo: Obrigatória

"declaracao_incerteza": "As incertezas expandidas de medição (U) relatadas são declaradas como a incerteza padrão combinada multiplicada pelo fator de abrangência k, que, para uma distribuição t, com um número efetivo de graus de liberdade v eff, o qual corresponde a uma probabilidade de abrangência de 95,45 %. A incerteza de medição expandida foi relatada de acordo com a publicação Avaliação de Dados de Medição - Guia para a Expressão de Incerteza de medição - GUM 2008."

Método de Medição

metodo_medicao

Array de strings com a descrição dos métodos de medição

Tipo: Obrigatória (o conteúdo é opcional)

Exemplo:
"metodo_medicao": [ "As entradas de tensão do instrumento padrão e do instrumento sob calibração foram conectadas em paralelo...", "Método de medição 2 teste" ]
metodo_medicao_equation

Array de strings com equações em formato LaTeX ou MathML

Tipo: Opcional

Exemplo:
"metodo_medicao_equation": [ "\delta_u = \dfrac{U_ac}{U_dc} {U_dc}" ]

Mensurando

Array com as definições do mensurando

Tipo: Obrigatório (pelo menos um)

Exemplo:
"mensurando": [ { "label": "acdc", "name": "Diferença AC-DC em tensão", "col_name": "\delta_u", "unit": "\micro\volt\volt\tothe{-1}", "unc_relativa": false, "refType": "basic_measuredValue" }, { "label": "rstdc", "name": "Resistência em Corrente Contínua", "col_name": "\m", "unit": "\ohm", "unc_relativa": true, "refType": "basic_measuredValue" } ]
Descrição do array:
Chave Descrição Tipo Exemplo
label Identificador do mensurando Obrigatória "label": "acdc"
name Título da tabela de resultados Obrigatória "name": "Diferença AC-DC em tensão"
col_name Nome da coluna com valores numéricos Obrigatória "col_name": "V.m."
unit Unidade do resultado (formato D-SI) Obrigatória "unit": "\micro\wolt\wolt\tothe{-1}"
unc_relativa Indica se a incerteza é relativa Obrigatória "unc_relativa": false
refType RefType do resultado Opcional (recomendado) "refType": "basic_measuredValue"
